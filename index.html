Browser Rendering Pipeline

Jab browser kisi webpage ko load karta hai, to HTML aur CSS ko parse karke screen par pixels banane tak ye steps follow hote hain:

HTML

HTML File

Browser .html file ko padhna shuru karta hai (Parsing).

↓ Tokenization

Tokens

HTML ko chhote-chhote tokens mein toda jata hai — jaise tags, attributes, text.

↓

DOM Tree

Tokens se ek tree banta hai, jisme har element ek node hota hai (parent-child relation ke saath).

CSS

CSS Rules

Browser saare CSS (inline, internal, external) ko parse karta hai.

↓ Parsing

CSSOM Tree

CSS rules se ek doosra tree banta hai, jisme har element ke final computed styles hote hain.

↓

Render Tree

DOM Tree + CSSOM Tree combine hote hain. Sirf wahi nodes is tree mein aate hain jo screen par dikhne wale hain (jaise display:none wale elements skip ho jaate hain). Yahi Render Tree aage Layout aur Paint ke liye base banta hai.
